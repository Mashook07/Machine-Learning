# Machine Learning - Learning Phase

This repository contains notebooks and datasets used during the learning phase of Machine Learning.

## Repository Contents

* **[ml.ipynb](ml.ipynb)**: A Jupyter Notebook containing machine learning code, models, and analysis.
* **[insurance.csv](insurance.csv)**: The dataset used for modeling (e.g., insurance charges prediction).

## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Mashook07/machine-learning.git
   ```
2. Open the notebook in VS Code, Jupyter, or Google Colab.
3. Run the cells to see the machine learning pipeline.
